# Crypto Exchange App



Простое приложение для отображения информации о криптовалютах с использованием React и RTK Query.

## Технологии

Проект использует следующие технологии:

- React / Next.js
- TypeScript
- SCSS modules
- RTK Query
- React Query
- React Crypto Icons
